Where Web Design Meets Cybersecurity

Web design and cybersecurity are connected because websites are one of the main ways people interact with technology. A website needs to be easy to use, but it also needs to be built carefully so that users and information are protected.

Learning HTML and forms helped me understand how users enter data into a website. From a cybersecurity point of view, that matters because every form is also a place where bad input could cause problems if the site is not protected correctly.

CSS helped me understand layout and design, while JavaScript showed me how a page can respond to user actions. Together, these skills make websites more interactive, but they also remind me that every feature should be built thoughtfully. The internet is useful, but it also has a lot of digital raccoons digging through the trash.
