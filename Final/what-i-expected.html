What I Expected

My original goal for this class was to take away a better understanding of HTML, CSS, and how websites are structured. I also wanted to become more comfortable using VS Code, Git, and creating webpages on my own.

Looking back, I feel like I did get what I wanted from this class. I learned how to build pages with proper HTML structure, add style with CSS, and use JavaScript to make a page interactive.

I also learned that web design is not just about making something look nice. It is about organizing information, making pages usable, and understanding how different files work together. Basically, I came for the HTML and stayed because CSS decided to test my patience.
